## Versioning Scheme

Deepak Labs uses [Semantic Versioning](https://semver.org/): `MAJOR.MINOR.PATCH[-prerelease]`.

| Segment      | Meaning                                                       |
| ------------ | ------------------------------------------------------------ |
| `MAJOR`      | Incompatible or product-defining changes.                    |
| `MINOR`      | Backwards-compatible functionality.                          |
| `PATCH`      | Backwards-compatible fixes.                                  |
| `prerelease` | Stability label: `-alpha`, `-beta`, `-rc`.                   |

## Current Stage

`alpha` — the product is under active foundation work and is not feature-complete or stable.
