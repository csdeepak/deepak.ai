# assets/

Static and design assets for Deepak Labs.

This directory holds non-code assets: images, icons, fonts, logos, design exports, and other media referenced by applications and documentation.

## Suggested Organization

| Subdirectory | Contents |
| --- | --- |
| `images/` | Photographs, screenshots, illustrations. |
| `icons/` | Icon sets and SVGs. |
| `fonts/` | Font files. |
| `brand/` | Logos and brand assets. |
| `design/` | Design-tool exports and references. |

## Conventions

- Optimize assets before committing (compress images, subset fonts).
- Use descriptive `kebab-case` filenames.
- Keep large binaries out of the repository where a better store exists; document the location if so.
