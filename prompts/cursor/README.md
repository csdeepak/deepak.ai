# prompts/cursor/

Reusable prompts for the **Cursor** editor.

## What Belongs Here

Prompts and rules used within Cursor: inline edit instructions, chat prompts, and `.cursor`-style rules that guide in-editor AI assistance for this repository.

## Conventions

- One prompt per Markdown file, `kebab-case.md`.
- Include: **intent**, the **prompt** itself, and **expected output**.
- Keep prompts reusable; project-specific context belongs in [`../../memory/`](../../memory).
- Note whether a prompt is meant for inline edits, chat, or project rules.
