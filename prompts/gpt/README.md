# prompts/gpt/

Reusable prompts for **OpenAI GPT** models.

## What Belongs Here

Prompts tailored to GPT models and GPT-based tools. Good candidates include general code generation, brainstorming, drafting, and tasks where GPT performs well.

## Conventions

- One prompt per Markdown file, `kebab-case.md`.
- Include: **intent**, the **prompt** itself, and **expected output**.
- Keep prompts reusable; project-specific context belongs in [`../../memory/`](../../memory).
- Note any model or configuration assumptions.
