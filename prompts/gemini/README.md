# prompts/gemini/

Reusable prompts for **Google Gemini** models.

## What Belongs Here

Prompts tailored to Gemini models and Gemini-based tools. Good candidates include multimodal tasks, long-context analysis, and research-oriented work.

## Conventions

- One prompt per Markdown file, `kebab-case.md`.
- Include: **intent**, the **prompt** itself, and **expected output**.
- Keep prompts reusable; project-specific context belongs in [`../../memory/`](../../memory).
- Note any model or configuration assumptions.
