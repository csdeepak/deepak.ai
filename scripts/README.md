# scripts/

Automation, build, and maintenance scripts.

This directory holds tooling that supports development and operations: setup helpers, code generation, data migration, documentation checks, and CI utilities.

## Conventions

- Each script includes a header comment describing its purpose, inputs, and outputs.
- Prefer cross-platform scripts; note platform requirements when unavoidable.
- Group related scripts in subdirectories as the collection grows.

No scripts have been created yet. They will be added as the tech stack and workflows are established in [`docs/06-TECH_STACK.md`](../docs/06-TECH_STACK.md).
